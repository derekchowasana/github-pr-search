Github PR Search Helper (GPSH)

Drag this link GPSH to your bookmarks bar.

How to use:

  1. Log into your Asana Github account
  2. Go to https://github.com/pulls
  3. Click the bookmark and search

The typeahead fetches from the https://github.com/orgs/Asana/people endpoint. To get full search results, you must be a member of the Asana org in Github.
